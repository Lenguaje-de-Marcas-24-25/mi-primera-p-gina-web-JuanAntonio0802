Definiciones y Listas

  1. W3C
    World Wide Web Consortium, principal organización para desarrollar estándares y protocolos web.
  2. WHATWG
    Web Hypertext Application Technology Working Group, grupo responsable de la evolución de HTML y varias APIs de navegador.
  3. HTML
    Lenguaje utilizado para estructurar y diseñar contenido en la web, fundamental para el desarrollo web.
  4. XHTML
    Extensible HyperText Markup Language, una variante de HTML con reglas más estrictas basadas en XML.
  5. DTD
    Document Type Definition, documento que especifica las reglas y estructura de archivos XML o HTML.
  6. APIs
    1. Drag and Drop
      Permite a los usuarios mover elementos en la interfaz mediante arrastrar y soltar.
    2. Geolocalización
      API que permite detectar la ubicación aproximada del usuario, útil en aplicaciones basadas en localización.
    3. Almacenamiento Local
      API que guarda datos de forma persistente en el navegador, manteniéndolos disponibles incluso al cerrar el navegador.
    4. App Cache
      Permite almacenar archivos de una aplicación web para que se puedan utilizar sin conexión a internet.
    5. Web Workers
      API que permite ejecutar procesos en segundo plano, mejorando la eficiencia de aplicaciones intensivas en cálculo.
  7. Editor WYSIWYG
    Un editor visual que permite ver el contenido mientras se edita, sin necesidad de escribir código HTML directamente.
  8. Navegadores
    1. Mozilla Firefox
      Desarrollado por Mozilla, conocido por su enfoque en la privacidad y su carácter de código abierto.
    2. Google Chrome
      Navegador de Google, famoso por su velocidad y amplio soporte de tecnologías web.
    3. Safari
      Navegador de Apple, optimizado especialmente para dispositivos de la misma marca.
    4. Opera
      Opera Software desarrolla este navegador, que destaca por integrar características avanzadas como VPN gratuita.
